Define Indicator

Please chose a fitting indicator for your new dataset. An indicator indicates what "thing" the data describes. In a Metric the dataset can be referenced via this indicator. If the list does not include a suitable indicator, create a new one. In addition provide the unit your values are in. An Indicator has to be a generic concept, independent from location (country, city) and time. For example Unemployment, Death Rate, Happiness.
